# Django-note-app

## Tools
* **Django**
* **Django REST Framework**
* **SQLite DB**
* **Bootstrap**

## What does it do :
* **Register** (sign up)
* **Login**
* **Create notes**
* **Edit notes**
* **Delete notes**
* **out put the notes table as json through REST framework** (launch the app and visit http://127.0.0.1:8000/restApiOutput/ to see json data)
